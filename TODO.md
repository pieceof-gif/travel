# 최고의 여행 — 업데이트 로드맵

## 📌 예정 기능

### 국내 여행 비교 모드
- **우선순위**: 낮음 (해외 비교 완성도 우선)
- **방향**: 해외와 같은 리스트가 아닌 **별도 탭/모드**로 분리
  ```
  [해외 여행] | [국내 여행]  ← 상단 탭 전환
  ```
- **비교 항목 차이점**:
  | 항목 | 해외 | 국내 |
  |------|------|------|
  | 교통 | 항공권 | KTX/버스/항공 |
  | 숙박 | 동일 | 동일 |
  | 현지 비용 | 동일 | 동일 |
  | 환전 | 표시 | 불필요 |
  | 출입국 준비 | 비자/여권 | 불필요 |
  | 안전도 | 외교부 경보 | 불필요 or 간소화 |
- **후보 여행지**: 부산, 강릉, 여수, 경주, (제주도는 기존 데이터 활용)
- **데이터 전략**: 하드코딩 + 예약 사이트 링크 (해외와 동일 방식)
  - 교통 요금: 정찰제라 하드코딩 충분 (KTX 서울→부산 59,800원 등)
  - 숙소: 하드코딩 + 야놀자/여기어때 링크
  - 향후 공공데이터포털 API 연동 검토 가능
